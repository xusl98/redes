1.2 Redes

Una red es la interconexión de un conjunto de dispositivos capaces de comunicarse

1.2.1 Criterios de redes

  1. Rendimiento: Se mide mediante tasa de transferencia efectiva o la latencia
  2. Fiabilidad: Exactitud de entrega y frecuencia de fallo
  3. Seguridad: Protección de datos frente a fallos y accesos no autorizados

1.2.2 Estructuras físicas

Atributos de una red
  • Tipo de conexión
    1. Punto a punto: Enlace dedicado entre dos dispositivos
    2. Multipunto: Varios dispositivos comparten enlace
  • Topología física: Forma en la que está diseñada la red físicamente
    1. Topología en Malla: Cada dispositivo tiene un enlace punto a punto y dedicado con cualquier otro dispositivo
      • Ventajas
        • Los enlaces dedicados transportan solo los datos propios
        • Es robusta, si un enlace falla no inhabilita el sistema
        • Privacidad / seguridad
        • Los enlaces punto a punto facilitan identificar los fallos
      • Desventajas
        • Debido a cantidad de cable, díficil instalación y reconfiguración
        • La masa de cables puede ser mayor que el espacio disponible
        • Puede ser prohibitivamente caro
    2. Topología en Estrella: cada dispositivo solamente tiene un enlacec punto a punto con el controlador central (llamado concentrador o hub), los dispositivos no están directamente enlazados entre sí
      • Ventajas
        • Fácil de instalar y configurar
        • MenosCable
        • Robustez, la desconexión de un cable solo afecta a un dispositivo
        • Fácil de identificar los fallos
      • Desventajas
        • Dependencia de todo el sistema del concentrador
        • Requiere mucho cale (no tanto como la de malla)
