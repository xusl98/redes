Tema 4 - Transmisión digital

4.1 Conversión Digital a Digital

4.1.1 Codificación de línea

Proceso de convertir datos digitales en señales digitales. Los datos de texto, números, imágenes audio o voz se almacenan en la memoria del ordenador en secuencias de bits. La codificación de línea convierte una secuencia de bits en una señal digital. En el emisor los datos se codifican y en el recepetor se decodifican.
Características:
  • Elemento de señal frente a elemento de datos: Un elemento de datos es la entidad más pequeña ¡, que puede representar una pieza de información, el bit. Un elemento de señal lleva elementos de datos y es la señal más corta de una señal digital.
    Elementos de datos: lo que necesitamos enviar.
    Elementos de señal: lo que podemos enviar.
  • Tasa de datos frente a tasa de señales: La tasa de datos define el número de elementos de datos (bits) enviados en 1 segundo (bps). La tasa ded señales es el número de elementos de señal enviados en 1 segundo (la unidad es el baudio).
    La tasa de datos se puede llamar tasa de bits y la tasa de señales se conoce también como tasa de pulsos, tasa de modulación o tasa de baudios.
    El objetivo es aumentar la tasa de datos mientras se reduce la de señales (Aumentar velocidad de transmisión y reducir ancho de banda requerido).

    Práctica:
  • Ancho de banda: Aunque el ancho de banda real de una señal digital es infinito, el ancho de banda efectivo es finito. Se puede decir que la tasa de baudios determina el ancho de banda requerido para una señal digital.

    Práctica:
  • Componentes DC y Autosincronización y Variaciones de Línea Base

    En una transmisión digital el reloj del receptor es 0.1% más rápido que el del emisor.
  • Detección de errores incorporada
  • Inmunidad al ruido y las interferencias

4.1.3 Codificación de bloques

Ofrece redundancia, cambia un bloque de m en un bloque de n bits, donde n es mayor que m (mB/nB).
  • 4B/5B
  • 8B/10B

4.1.4 Aleatorización

Una larga secuencia de 0 provoca problemas de sincronización, una solución es la aleatorización.
  • B8ZS: Bipolarización con sustitución de 8 ceros, se suele usar en norteamérica. Ocho voltajes a nivel cero se sustituyen por 000VB0VB.
  • HDB3: Bipolar de alta densidad con 3 ceros, se suele usar fuera de norteamérica. Cuatro voltajes a nivel cero se sustituyen por 000V o B00V.

4.2 Conversión Analógica a Digital

4.2.1 Modulación por codificación de pulsos (PCM)

Es la técnica más habitual, tiene 3 procesos:
  • Muestreo: La señal analógica es muestreada cada Ts donde Ts es el intervalo de muestreo o periodo. El inverso del intervalo de muestreo se denomina tasa de muestreo. El método de muestreo más común es el muestreo de cresta plana (de muestreo y retención). El muestreo también es conocido como modulación por amplitud de pulsos (PAM).
    Según el teorema de Nyquist, la tasa de muestreo debe ser al menos dos veces la frecuencia más alta contenida en la señal.
    pag116
  • Cuantificación: El resultado del muestreo es una serie de pulsos con valores de amplitud comprendidos entre la mínima y la máxima amplitud de la señal, etapas de la cuantificación:
    1. Se asume que la señal analógica tiene amplitudes instantáneas entre Vmin y Vmax
    2. Se divide el rango en L zonas, cada una de ancho delta => delta = (Vmax - 2Vmin) / L
    3. Se asignan valores cuantificados de 0 a L - 1 en el punto medio de cada zona
    4. Se aproxima el valor de la amplitud de la muestra a los valores cuantificados

4.2.2 Modulación Delta (DM)

Es la técnica de modulación más simple. PCM encuentra el valor de la amplitud de la señal de cada muestra, DM encuentra el cambio respecto a la muestra anterior. Modulador
Se utiliza un modulador en el emisor para crear un flujo de bits a partir de la señal analógica. El proceso registra los cambios, que son denominados deltas.
Demodulador
Toma la señal de datos, y utilizando un constructor de señal de escaler y la unidad de retardo, crea la señal analógica.
DM Adaptativa
Se obtienen mejores prestaciones si el valor de delta no es fijo. En este tipo el valor de delta cambia de acuerdo a la amplitud de la señal analógica.
Error de cuantificación
En DM siempre se introduce un error de cuantificación, pero es mucho menor que para PCM.

4.3 Modos de Transmisión

4.3.1 Transmisión paralela

Se envían n bits agrupados al mismo tiempo en lugar de uno solo. Se usan n hilos para enviar n bits cada vez, de esta forma cada bit tiene su propio hilo y todos los n bits de un grupo se pueden transmitir en un solo pulso de reloj.
La ventaja de este tipo de transmisión es la velocidad, pero se encarece bastante por lo tanto se usa habitualmente para distancias cortas.

4.3.2 Transmisión en serie

Un bit sigue a otro por lo que solo se necesita un canal de comunicación.
La ventaja de esta forma es que al tener un solo canal de comunicación se reduce el coste aproximadamente en n.
La transmisión en serie puede llevarse a cabo de tres maneras.
  1. Transmisión asíncrona: Se denomina así debido a que la temporización de la señal no es importante.
    Para avisar al receptor de la llegada de un nuevo grupo se añade un bit extra al principio de cada byte, sule ser 0 y se denomina bit de inicio. Para permitir al recepetor saber que el byte ha terminado se añaden uno o varios bits adicionales al final de cada byte, se llaman bits de parada.
    Puede haber un intervalo entre cada byte.
    Asíncrono aquí significa a nivel de byte, pero los bits siguen sincronizados, su duración es la misma.
  2. Transmisión síncrona: Se envía un bit detrás de otro, sin bits de inicio/parada o intervalos. Es responsabilidad del receptor agrupar los bits.
  3. Isócrona: Garantiza que los datos lleguen a una tasa fija, por eso se usa para vídeo y audio en tiempo real.
